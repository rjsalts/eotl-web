Games Within the Game

Here is a list of games that are available for play when you log on the game of EotL. Most of the following games can be found in the Lounge's Vending Machine at little or no charge!

  • B U G H O U S E (Binford Chess)
  • Number Puzzle (web version written in Java)
  • Connect Four
  • Hearts
  • Spades
  • Bridge
  • Blackjack
  • Gobang
  • Othello
  • Stratego
  • The Letter Game
  • Uno
  • Poker Solitaire
  • Word Association Game (wag)
  • Mastermind
  • Minesweeper
  • The Cool Game

There used to be a Towers of Hanoi puzzle that gave out experience for solving it, but too many players used scripts to solve it in a matter of seconds. What fun is that, I ask you?!?
