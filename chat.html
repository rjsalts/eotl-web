First of all, you need a Java-enabled browser to use the chat room.

From http://www.sneakerchat.com/Instructions.html:

When the applet has finished loading, you will see a login form. This form has five fields for you to fill in. Only the first three are required.

The Image URL feature will not work for anyone who is not a member of SneakerChat. However, if you are a member, you can type in the URL to your image on the SneakerChat server. (the image must reside on our server for it to be displayed.)

Press Enter or click on the Connect to Server button in order to initiate the connection.

Also, the Chat will not work if you're behind a firewall. Once you get in, you should be able to figure out how to navigate the chat room.

Internet Explorer users be warned: The chat applet will give you strange Javascript errors.
